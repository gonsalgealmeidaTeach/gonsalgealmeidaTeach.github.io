# Week 8 – Final Project

This project integrates multivariate statistical methods:

- Correlation analysis
- Principal Component Analysis (PCA)
- Regression modeling
- Clustering
- Portfolio optimization

## Reproducibility

1. Run setup/install_packages.R
2. Ensure data is in /data folder
3. Run session08_final_integration.R

## Output

- Portfolio weights
- Risk-return profile
- Multivariate insights for decision-making

  IMPORTANT:

1. Run session08_generate_portfolio_data.R FIRST
2. This creates the dataset required for Week 8
3. Do NOT modify file paths
4. Ensure internet connection for initial download
