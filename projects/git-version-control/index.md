## Why Version Control?

Graduate-level work involves:

- Reproducible research
- Collaborative projects
- Iterative model development
- Code tracking and documentation
- Professional portfolio development

Git allows you to:

- Track every change in your work
- Revert to previous versions
- Collaborate safely
- Maintain structured research workflows
- Build a professional GitHub portfolio
