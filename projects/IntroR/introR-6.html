Overview

Base R graphics provide quick, dependable plotting for exploratory analysis in STAT 312/312R. Below we use base functions to visualize univariate samples (histograms, boxplots, QQ plots) and bivariate relationships (scatterplots, line plots), mirroring early chapters in Engineering Statistics (Montgomery, Runger, Hubele).

Sample dataset: thickness.csv (stored at /projects/datasets/thickness.csv). In earlier pages we assumed a column named value.

0) Load a Sample

# Optional: read a shared dataset if available
# install.packages("readr")  # if needed
library(readr)
thick <- read_csv("../datasets/thickness.csv")$value  # numeric vector
length(thick); summary(thick)

If you do not have the CSV, you may substitute your own numeric vector.

1) Scatterplots

# Simple x-y scatter (example with synthetic data)
set.seed(312)
x <- 1:30
y <- 2 + 0.1*x + rnorm(30, sd = 0.2)

plot(x, y,
     main = "Process Response vs. Index",
     xlab = "Order / Time Index", ylab = "Response",
     pch = 19)
abline(lm(y ~ x))        # add least-squares line
legend("topleft", legend = c("Observations","Fitted line"),
       pch = c(19, NA), lty = c(0,1), bty = "n")

Use pch for point symbol, cex for size, and abline() to overlay reference lines (fits, means, or specification limits).

2) Histograms and Density Curves

# Histogram of thickness measurements
hist(thick, breaks = "Sturges",
     main = "Thickness Distribution",
     xlab = "Thickness (mm)")

# Overlay a kernel density estimate
dens <- density(thick, na.rm = TRUE)
lines(dens, lwd = 2)

Histogram binning may affect interpretation; adjust breaks and compare shapes.

3) Boxplots (with or without groups)

# Single-sample boxplot
boxplot(thick, horizontal = TRUE,
        main = "Boxplot of Thickness",
        xlab = "mm")

# Example: grouped boxplots (synthetic groups A/B)
grp <- rep(c("A","B"), each = length(thick)/2)[1:length(thick)]
boxplot(thick ~ grp, main = "Thickness by Group", xlab = "Group", ylab = "mm")

Boxplots emphasize median, quartiles, and outliers—useful for comparing process conditions or lots.

4) Normal Q–Q Plots

# Assess approximate normality (e.g., for t-based intervals/tests)
qqnorm(thick, main = "Normal Q-Q Plot: Thickness")
qqline(thick)

Points close to the line suggest an approximately normal distribution—an assumption often used in interval estimation and hypothesis testing.

5) Line Plots (Order/Time Series)

# Plot in order collected (useful to spot drift or cycles)
plot(thick, type = "l",
     main = "Thickness Over Order/Time",
     xlab = "Index", ylab = "Thickness (mm)")
abline(h = mean(thick, na.rm = TRUE), lty = 2)

Use line plots to detect trends, shifts, or unusual runs before formal modeling.

6) Multipanel Layouts

# Arrange multiple panels, then restore original settings
op <- par(mfrow = c(2,2))

hist(thick, main = "Histogram", xlab = "mm")
boxplot(thick, horizontal = TRUE, main = "Boxplot", xlab = "mm")
plot(thick, type = "l", main = "Line Plot", xlab = "Index", ylab = "mm")
qqnorm(thick, main = "Q-Q Plot"); qqline(thick)

par(op)  # restore previous graphics parameters

Multipanel views help compare perspectives quickly during EDA.

7) Saving Figures to File

# PNG export (similar functions: pdf(), svg(), jpeg(), tiff())
png("thickness_histogram.png", width = 800, height = 600, res = 120)
hist(thick, breaks = 12, main = "Thickness", xlab = "mm")
dev.off()

Save plots to include in lab reports or R Markdown documents.

8) Mini Example (Aligned with the Text)

For a sample of thickness measurements, summarize center/spread, check normality, and visualize the distribution. Download data: thickness.csv. 

# Summary, histogram with density, and Q-Q plot
mx <- mean(thick, na.rm = TRUE); sx <- sd(thick, na.rm = TRUE)
c(mean = mx, sd = sx)

hist(thick, breaks = 10, main = "Thickness (mm)", xlab = "mm")
lines(density(thick, na.rm = TRUE), lwd = 2)

qqnorm(thick); qqline(thick)

These visuals support decisions about modeling assumptions and specification conformance.
