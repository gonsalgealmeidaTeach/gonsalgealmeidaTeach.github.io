How to Install R & RStudio

Department of Mathematics, Applied Mathematics, and Statistics
Case Western Reserve University

Steps

  1. Install R. Download the latest installer for your OS from CRAN and complete the setup.
  2. Install RStudio Desktop. Download the free RStudio (Posit) Desktop for your OS and install it.
  3. Open RStudio and verify versions. 
    R.version.string
R.version$platform
  4. Install essential packages. 
    install.packages(c("tidyverse","ggplot2","dplyr","readr","knitr","rmarkdown"))
  5. Optional: test R Markdown. 
    rmarkdown::draft("test", template = "html_document", package = "rmarkdown")
