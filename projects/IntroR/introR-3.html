1) What Are Packages?

Packages extend base R with additional functions, datasets, and documentation. In STAT 312/312R we will use packages for data import, visualization, and inference. Typical examples include tidyverse, ggplot2, dplyr, readr, and broom. 

# Install once per machine (Internet required)
install.packages(c("tidyverse","broom","readr"))

# Load each R session as needed
library(tidyverse)   # loads ggplot2, dplyr, readr, etc.
library(broom)       # tidying model output

If a function is not found (e.g., could not find function), confirm that the package is both installed and loaded.

2) Updating and Checking Packages

# Update installed packages (choose a CRAN mirror if prompted)
update.packages(ask = FALSE)

# List installed packages
head(installed.packages()[, c("Package","Version")])

# Load safely (returns TRUE/FALSE instead of stopping)
require(ggplot2)   # similar to library(), but quieter

Tip: If you see errors about permissions, restart RStudio and try again, or run as administrator (Windows).

3) Working Directory

The working directory (WD) is the folder where R reads and writes files by default. Set the WD to the course folder that contains your data and scripts. Avoid spaces and special characters in folder names. 

# Show current working directory
getwd()

# Set working directory (Windows example)
setwd("C:/Users/yourname/Documents/STAT312")

# macOS/Linux example (note forward slashes)
setwd("/Users/yourname/Documents/STAT312")

Best practice: Use an RStudio Project for each course or assignment. Open the .Rproj file and use relative paths; this removes the need to call setwd().

4) Mini Example (with Download)

Download the sample data used below: thickness.csv (path in repo: /projects/datasets/thickness.csv). 

library(readr); library(dplyr); library(ggplot2)

# If your script is in projects/R/, use the relative path:
thick <- read_csv("../datasets/thickness.csv")

# If your working directory is the site root, use:
# thick <- read_csv("projects/datasets/thickness.csv")

# Summaries
thick %>% summarize(n = n(),
                    mean = mean(value),
                    sd = sd(value),
                    q25 = quantile(value, .25),
                    median = median(value),
                    q75 = quantile(value, .75))

# Plot
ggplot(thick, aes(x = value)) +
  geom_histogram(bins = 12) +
  labs(x = "Thickness (mm)", y = "Count",
       title = "Process Thickness Distribution")

This mirrors the early chapters of Engineering Statistics: compute center/spread and visualize variability.
