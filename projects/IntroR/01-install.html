1. Install R & RStudio

CWRU · MAMS · STAT 312/312R Supplements

Goals

  • Install the R language and the RStudio IDE.
  • Run a first command and confirm your setup.

Steps

  1. Install R from CRAN.
  2. Install RStudio Desktop (free) from Posit.
  3. Open RStudio → Console tab → run: 
    version
sessionInfo()
2 + 2

Try it

  • Create an R script (File → New File → R Script) and save it as hello.R.
  • Put print("Hello, STAT 312!") in it and click “Run”.
